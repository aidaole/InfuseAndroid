# InfuseAndroid

这是一个电视播放器, 功能类似于IOS系统中的Infuse

这个应用的主要功能:

1. 使用Android平台, 适配TV
2. 支持SMB协议, 可以扫描和连接局域网中的SMB服务器
3. 支持扫描服务器中的视频文件, 主要分为电影(单文件), 电视剧(文件夹+季+单集)
4. 支持展示电影,电视剧的海报墙
5. 适配播放功能

编码要求:

1. 使用Android推荐的MVVM框架, 使用Jetpack现代库
2. UI部分使用compose编写
3. 使用协程, flow做状态同步和管理
4. 代码尽量简洁易懂, 具有合理的拆分和耦合
5. 可以使用hilt做依赖注入
